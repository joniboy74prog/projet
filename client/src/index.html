Better Food


Introduction

Better Food est un projet écrit en TypeScript (client) et Java (serveur), ayant pour but de capturer un code barre EAN.
Le code barre va alors être envoyé à l'aide d'un Websocket vers la partie Java.
La partie Java va alors récupérer les informations alimentaires depuis OpenFoodFact pour les afficher ensuite ici.
Pour cela nous utilisons le librairies suivantes :
- QuaggaJs pour la lecture d'un code barre
- Tyrus pour la mise en place du Websocket
- Javax.json pour le traitement des donneés dans un fichier JSON
